Въведение

Какво е Bootstrap?

Bootstrap е безплатен, отворен CSS framework, създаден от Twitter.
Той предоставя готови компоненти, grid система и JavaScript плъгини, с които можете да изграждате responsive уебсайтове бързо и лесно — без да пишете CSS от нулата.

История

История на Bootstrap

  • 2011Създаден от Mark Otto и Jacob Thornton в Twitter за вътрешна употреба.
  • 2011Пуснат като open-source проект в GitHub под MIT лиценз.
  • 2013Bootstrap 3 — въвежда mobile-first подход и нова grid система.
  • 2018Bootstrap 4 — пълно пренаписване с Flexbox, нови утилити класове.
  • 2021Bootstrap 5 — без jQuery, подобрен grid (CSS Grid/Flexbox), нови компоненти. Текуща версия.

Включване

Как се включва Bootstrap?

Само два реда в HTML файла — CSS в <head> и JS преди </body>:

<!-- CSS в <head> -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"
      rel="stylesheet">

<!-- JS преди </body> -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Алтернативно: npm install bootstrap за npm проекти.

Grid система

Grid система

Bootstrap разделя реда на 12 колони. Използвайте класове за различни размери на екрана:

<div class="container">
  <div class="row g-4">

    <!-- Заема 12/12 на мобилен, 6/12 на таблет, 4/12 на десктоп -->
    <div class="col-12 col-md-6 col-lg-4">Стая 1</div>
    <div class="col-12 col-md-6 col-lg-4">Стая 2</div>
    <div class="col-12 col-md-6 col-lg-4">Стая 3</div>

  </div>
</div>

Компонент – Navbar

Navbar в Хотела

Bootstrap Navbar автоматично се свива в hamburger меню на малки екрани:

<nav class="navbar navbar-expand-lg">
  <div class="container">

    <a class="navbar-brand" href="index.html">🏨 Ел Кастела</a>

    <!-- Бутон за мобилно меню -->
    <button class="navbar-toggler" data-bs-toggle="collapse"
            data-bs-target="#mainNav"></button>

    <div class="collapse navbar-collapse" id="mainNav">
      <!-- Навигационни линкове -->
    </div>

  </div>
</nav>

Bootstrap Cards — Стаите

Компонент – Forms

Bootstrap Форми

Формата за резервация и регистрация използват Bootstrap form класове:

<!-- Регистрационна форма -->
<form>
  <div class="mb-3">
    <label class="form-label fw-semibold">Имейл адрес</label>
    <input type="email" class="form-control" required>
  </div>

  <div class="form-check mb-3">
    <input type="checkbox" class="form-check-input">
    <label class="form-check-label">Приемам условията</label>
  </div>

  <button class="btn btn-primary w-100">Регистрирай се</button>
</form>

Оценка

Предимства и Недостатъци

✔ Предимства
  • ⚡ Бърза разработка — готови компоненти
  • 📱 Mobile-first responsive по подразбиране
  • 🌐 Огромна общност и документация
  • 🔌 Не изисква jQuery (Bootstrap 5)
  • 🎨 Лесна персонализация с CSS променливи
  • ✅ Кросбраузърна съвместимост
✖ Недостатъци
  • 📦 По-голям файл (~22KB CSS minified)
  • 🎭 Сайтовете изглеждат подобно без персонализация
  • 📚 Изисква учене на Bootstrap класове
  • ⚙️ Прекомерен код при малки проекти
  • 🔧 По-малко контрол спрямо чист CSS
